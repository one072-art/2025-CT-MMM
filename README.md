# 2025-CT-MMM
option
🐟 야옹! 매지호를 부탁해
매지호 수호신 곤지 – 환경 스토리 기반 낚시 어드벤처 게임

“호수가 이상해졌어… 내 말이 들린다면, 도와줘.”
말하는 고양이 수호신 곤지와 함께 캠퍼스 호수의 이상 현상을 해결하는
스토리 중심 2D 낚시 어드벤처 게임입니다.

🎮 게임 개요

장르: 스토리 기반 2D 낚시 / 어드벤처

플랫폼: PC (Python / Pygame)

플레이 시간: 약 15~25분

특징

환경 오염을 테마로 한 스토리 진행

QTE 기반 낚시 시스템

단계별 스테이지 & 도감 수집

보스전(최종 스테이지) 포함

📖 스토리

최근 캠퍼스의 매지호에서 이상 현상이 발생한다.
물빛은 탁해지고, 물고기들은 변이를 일으킨다.

플레이어는 우연히
호수의 수호신 **말하는 고양이 ‘곤지’**의 목소리를 듣게 되고,
환경보건센터의 박사와 함께
호수의 오염 원인과 정체불명의 ‘교란 에너지’를 추적하게 된다.

호숫가 → 정자 → 노천극장 뒤 → 거북섬
점점 심각해지는 변이의 끝에는
매지호의 폭군이 기다리고 있다.

🗺️ 스테이지 구성
스테이지	장소	특징
STAGE 1	호숫가	기본 낚시 튜토리얼
STAGE 2	정자	강화 낚싯대 획득
STAGE 3	노천극장 뒤	외래종 & 난이도 상승
STAGE 4	거북섬	보스전 (Hard Mode)

각 스테이지마다 모든 물고기를 1마리 이상 포획해야 다음 단계로 진행할 수 있습니다.

🎣 게임 플레이 방법
기본 조작
기능	조작
이동	방향키 / WASD
상호작용	SPACE
낚시 던지기	마우스 왼쪽 버튼 길게 누른 후 떼기
훅셋	! 표시가 뜰 때 클릭
도감 보기	C
로비 이동	ESC
낚시 시스템

캐스팅: 마우스를 꾹 눌러 힘 조절

입질 대기: 찌가 흔들리면 집중

훅셋 QTE: 타이밍에 맞게 클릭

릴링: 장력 조절하며 끌어올리기

⚠️ 장력이 100%가 되면 줄이 끊어집니다!

👹 보스전 (STAGE 4)

SPACE 연타로 보스를 끌어당기기

보스가 광분 상태일 때는 공격 금지

광분 중 공격 시 오히려 보스가 회복함

타이밍과 인내심이 핵심입니다.

📚 도감 시스템

각 스테이지별 물고기 도감 제공

아직 잡지 못한 물고기는 실루엣으로 표시

도감 완성이 곧 스테이지 클리어 조건

🛠️ 실행 방법
1. 환경 준비
pip install pygame

2. 실행
python final_project.py


⚠️ 이미지 / 사운드 폴더 구조를 그대로 유지해야 정상 실행됩니다.
